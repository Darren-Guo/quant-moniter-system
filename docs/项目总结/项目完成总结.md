## 🚀 项目技术亮点

### 1. 智能自适应系统

## 🛠️ 快速启动指南

### 1. 环境准备
```bash
# 克隆项目
git clone https://github.com/Darren-Guo/quant-moniter-system.git
cd quant-moniter-system

# 创建虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动服务
```bash
# 启动Web应用
python web_app.py
# 或使用启动脚本
./start_web.sh

# 访问Web界面
# http://localhost:8080
```

### 3. API测试
```bash
# 系统状态
curl http://localhost:8080/api/status

# 市场数据
curl http://localhost:8080/api/market-data

# 智能刷新统计
curl http://localhost:8080/api/refresh-stats
```
