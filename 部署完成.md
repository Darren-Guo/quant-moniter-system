## 🎮 使用说明

### 1. 访问Web界面
打开浏览器访问：http://localhost:8080

### 2. 开始监控
1. 在输入框中输入股票代码（用逗号分隔）
   ```
   9988.HK, 1810.HK, AAPL, NVDA, XPEV, MI
   ```
2. 点击"开始监控"按钮
3. 查看实时股票数据

### 3. 功能操作
- **查看股票详情**: 点击股票卡片
- **查看告警**: 查看告警面板
- **停止监控**: 点击停止按钮
- **修改监控列表**: 重新输入股票代码

## 🔧 API接口

### REST API
- `GET /api/status` - 系统状态
- `GET /api/market-data` - 市场数据
- `GET /api/alerts` - 告警信息
- `POST /api/start-monitoring` - 开始监控
- `POST /api/stop-monitoring` - 停止监控

### 测试API
```bash
# 检查系统状态
curl http://localhost:8080/api/status

# 获取市场数据
curl http://localhost:8080/api/market-data

# 开始监控
curl -X POST http://localhost:8080/api/start-monitoring \
  -H "Content-Type: application/json" \
  -d '{"symbols": ["AAPL", "NVDA"]}'
```

## 🔍 故障排除

### 常见问题
1. **无法访问 http://localhost:8080**
   ```bash
   # 检查端口占用
   netstat -tlnp | grep 8080
   
   # 检查服务状态
   ps aux | grep start_simple.py
   ```

2. **页面加载错误**
   - 检查浏览器控制台 (F12)
   - 查看服务器日志
   - 清除浏览器缓存

3. **数据不更新**
   - 检查WebSocket连接
   - 确认监控已启动
   - 刷新页面

### 重启服务
```bash
# 停止当前服务
pkill -f start_simple.py

# 重新启动
cd /root/.openclaw/workspace/quant_monitor
python3 start_simple.py
```
